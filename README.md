# 🍽️ Douaa Gourmandises

تطبيق متكامل لطلب الطعام مع تكاملات متقدمة

## المزايا الرئيسية

✅ تسجيل دخول وتسجيل جديد
✅ تصفح المطاعم والقوائم
✅ عربة التسوق والطلب
✅ نظام الدفع الآمن
✅ تتبع الطلب على الخريطة
✅ تقييم المطاعم والوجبات
✅ تكامل واتس آب
✅ لوحة تحكم للمطاعم
✅ إشعارات فورية
✅ ملف شخصي للمستخدمين

## التقنيات المستخدمة

### 📱 Mobile (Frontend)
- **Flutter** - تطبيق موبايل متقاطع المنصات
- **Provider** - إدارة الحالة
- **Dio** - HTTP Client

### 🖥️ Backend
- **Node.js** - Runtime Environment
- **Express.js** - Web Framework
- **PostgreSQL** - Database
- **JWT** - Authentication
- **Socket.io** - Real-time Updates
- **Stripe** - Payment Integration
- **WhatsApp Business API** - WhatsApp Integration

## البنية المشروع

```
douaa-gourmandises/
├── mobile/              # تطبيق Flutter
├── backend/             # خادم Node.js
├── database/            # PostgreSQL Scripts
└── docs/                # التوثيق
```

## البدء السريع

### الموبايل
```bash
cd mobile
flutter pub get
flutter run
```

### الخادم
```bash
cd backend
npm install
npm start
```

## المتطلبات

- Flutter 3.0+
- Node.js 16+
- PostgreSQL 12+
- Dart 3.0+
